# Source Registry

This registry tracks the comparative formation-document corpus. It records what has been added, how it is retained, where the canonical text lives, and whether alignment work is complete.

Status values:

- `registered`: source is recognized but not yet analyzed
- `drafted`: text or metadata is present and an alignment memo exists in draft form
- `complete`: source, metadata, and alignment memo are present

## Notes

- The registry records corpus status, not interpretive certainty.
- `complete` means the repository contains enough retained text, metadata, and analysis to support cross-referencing against `PRINCIPLES.md`.
- Several multilingual sources still require native-language review even though a first-pass memo is present.
